# Errors & Diagnostics

## Policy
- Errors are first-class citizens
- No hiding errors behind logs-only

## Error Center
- categories: Printer/Klippy, Moonraker, Host/Services, Network, UI
- each entry: source, timestamp, severity, summary, details, suggested action

## Support Bundle
- must be reproducible
- includes last logs, service states, versions, ports, configs snapshot (redacted)
